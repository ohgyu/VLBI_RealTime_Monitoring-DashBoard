# VLBI Real-Time Monitoring Dashboard

실시간 VLBI 시스템 상태를 모니터링하기 위한 Python 기반 대시보드 프로그램입니다.

PyQt6 GUI와 Matplotlib 그래프를 사용하여 장비 상태 및 센서 데이터를 시각화합니다.

---

## Features

- 실시간 장비 상태 모니터링
- 센서 데이터 시각화 (Matplotlib)
- 온도 및 장비 상태 알람 기능
- Threshold 기반 경고 시스템
- SQLite 데이터 저장

---

## Tech Stack

- Python
- PyQt6
- Matplotlib
- SQLite
